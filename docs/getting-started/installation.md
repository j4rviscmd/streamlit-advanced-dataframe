# Installation

## Requirements

- Python 3.12+
- Streamlit 1.52+
- pandas 2.0+

## Install from PyPI

```bash
pip install streamlit-advanced-dataframe
```

## Install from source

```bash
git clone https://github.com/j4rviscmd/streamlit-advanced-dataframe.git
cd streamlit-advanced-dataframe
pip install -e .
```

## Verify installation

```python
import streamlit as st
from streamlit_advanced_dataframe import advanced_dataframe

st.write("Installation successful!")
```

Run with:

```bash
streamlit run your_app.py
```
