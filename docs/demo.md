## Try These Features

1. **Sorting** - Click any column header
2. **Resize Columns** - Drag the column borders
3. **Reorder Columns** - Drag and drop column headers
4. **Search** - Click the search icon in the toolbar
5. **Filter** - Click the filter icon on filterable columns
6. **Select Rows** - Click rows to select (in selection-enabled tables)
7. **Expand Rows** - Click the arrow icons (in expandable tables)
